Lab 3 extra system examples

`3/main.py` now works the same way as `2/main.py`: one task is described directly in the file by `PROBLEM_NAME`, `f`, and `exact`.

Files:
- `damped_oscillator.txt`: parameters for Example 5 from `3/examples.ipynb`.
- `jordan_chain_3.txt`: parameters for Example 6 from `3/examples.ipynb`.

Run from the repository root:

```bash
~/myenv/bin/python 3/main.py -i 3/input_examples/jordan_chain_3.txt
```

For another example, copy the matching `PROBLEM_NAME`, `f`, and `exact` block from `3/examples.ipynb` into `3/main.py`, then run with the corresponding input file.
